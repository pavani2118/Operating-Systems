## ✅ 1. Hello World

📄 *Filename*: `myfirstprogram.c`

```c
#include <stdio.h>

int main() {
    printf("Hello World!\n");
    return 0;
}
````

### ▶️ Compile and Run

```bash
gcc myfirstprogram.c -o myfirstprogram
./myfirstprogram
```
