## 🚀 How to Run

1. Make scripts executable

   ```bash
   chmod +x *.sh
   ```

2. Run a script

   ```bash
   ./multiplicationTable.sh
   ./pattern.sh
   ./sumFibonacci.sh
   ./sumPrime.sh
   ```
